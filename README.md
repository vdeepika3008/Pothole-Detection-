
# Pothole Detection System using Raspberry Pi

## Introduction
This project detects potholes using Raspberry Pi and camera.
It gives alert using LED and buzzer.

## Components Used
- Raspberry Pi
- Camera Module
- MPU6050 Sensor
- LED
- Buzzer
- Jumper Wires

## Working
1. Camera captures road image
2. Program detects pothole
3. Sensor detects vibration
4. LED turns ON
5. Buzzer sounds

## Result
Potholes are detected and alert is given.

## Conclusion
This project helps detect potholes and prevent accidents.
