
#Pothole Detection System using Raspberry Pi
#Introduction
This project is designed to detect potholes on roads using Raspberry Pi and a camera module. When a pothole is detected, the system gives an alert using an LED and buzzer. This helps in preventing accidents and improving road safety.
#Components Used
Raspberry Pi
Camera Module
MPU6050 Sensor
LED
Buzzer
Jumper Wires
#Working
The camera captures the road image.
The program processes the image.
The system detects potholes.
The MPU6050 sensor detects vibration.
The LED turns ON.
The buzzer gives an alert sound.
#Result
The system successfully detects potholes and alerts the user using LED and buzzer.
#Conclusion
This project helps in detecting potholes on roads and can be used to improve road safety and maintenance.
